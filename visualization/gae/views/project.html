How was it Rated
As part of our data collection lot of datas like number of forks,stargazers count, watchers count, stackoverflow data,social media data is collected. All metrics being used to rank and score OSS projects are normalized on a 0-1 scale. This is done because different metrics have different magnitudes. Then, projects are scored by adding all of the normalized metrics together.Finally, projects are ranked within OSS categories using the scores generated in the previous step.
